Couteau suisse pour QGIS

Cette extension est conçu comme une collection d'outils simples. Une sorte de "couteau suisse" pour QGIS.
La liste des outils proposés devrait s'allonger au cours du temps :

Dans Traitement > Boîte à outils > Assistant de styles

  • Style catégorisé à partir d'un CSV : utiliser un fichier CSV (avec les valeurs, libellés et couleurs) pour donner un style catégorisé à une couche vecteur
    Contenu du CSV: valeurs (du champ) ; libellés (des catégories) ; couleurs
    Exemple :
       BL;blé;#eee22c
       OR;orge;#47a266
    Choisir la couche cible et indiquer le champ à utiliser pour les catégories

Dans le menu Extensions > Couteau suisse pour QGIS

  • Inspecter les 1ères lignes d'un GROS fichier CSV ou texte : affichage très rapide.
  • Convertir du JSON en CSV : mettre à plat une arborescence json.
    Si le 1er niveau du JSON est une liste, chaque élément de celle-ci donnera 1 ligne du CSV
    Le contenu de chaque élément (et sous-éléments) sera transformé en colonnes du CSV
